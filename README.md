<h2>Variable & Data Types</h2>

> Python has five standard data types:
* Numbers
* String
* List
* Tuple
* Dictionary
* Text Type: 	str
* Numeric Types: 	int, float, complex
* Sequence Types: 	list, tuple, range
* Mapping Type: 	dict
* Set Types: 	set, frozenset
* Boolean Type: 	bool
* Binary Types: 	bytes, bytearray, memoryview
